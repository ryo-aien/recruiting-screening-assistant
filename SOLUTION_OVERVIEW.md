# 採用書類選考支援システム - 課題と解決アプローチ

## 解決したい課題

| 課題 | 影響 |
|------|------|
| 書類選考に時間がかかる | 採用担当者の負担増、他業務を圧迫 |
| 応募から判定までが遅い | 優秀な候補者が他社に流れる |
| 評価基準が担当者でバラつく | 選考の公平性・一貫性が保てない |
| 判断の根拠が残らない | 後から振り返りや改善ができない |

---

## 解決アプローチ

### 1. AIで「読む・整理する」を自動化

- 履歴書・職務経歴書を自動で読み取り
- スキル・経験年数・強み・懸念点を抽出
- 求人要件との適合度を0〜100点でスコア化

### 2. 判断は人間が行う

- AIは「優先順位をつける」だけ
- 合否の最終判断は必ず採用担当者が行う
- スコアの根拠（なぜこの点数か）を常に表示

### 3. 評価基準を統一

- 全候補者を同じ基準で評価
- 必須要件を満たさない場合は上位に来ない仕組み
- 設定は管理画面から調整可能

---

## 期待される効果

| 効果 | 内容 |
|------|------|
| 時間短縮 | 1件あたりのレビュー時間を削減 |
| 滞留解消 | 応募から初回判定までを短縮 |
| 一貫性向上 | 誰が見ても同じ基準で評価 |
| 説明可能 | なぜその評価かを常に確認できる |

---

## AIの使い方（方針）

- **やること**: 情報抽出、スコア計算、優先順位付け、根拠の提示
- **やらないこと**: 合否の自動決定、年齢・性別などによる判断
